# Carousels with CSS

A small demo repository showing how to build accessible, performant carousels using only CSS.

## Features

- Pure CSS carousel examples
- Focus and accessibility considerations
- Lightweight and responsive

## Usage

Open the example HTML files in your browser. No build step required.
